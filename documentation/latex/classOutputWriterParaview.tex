\doxysubsection{Detailed Description}
Write \texorpdfstring{$\ast$}{*}.vti files that can be viewed with Para\+View. 

The mesh that can be visualized in Para\+View corresponds to the mesh of the computational domain. All values are given for the nodes of the mesh, i.\+e., the corners of each cell. This means, values will be interpolated because the values are stored at positions given by the staggered grid.
